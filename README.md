# Client Cookies
Установка:
1. Установить файлы на сервер
2. Прописать в databases.cfg блок cookies

sqlite:
```
	"cookies"
	{
		"driver"			"sqlite"
		"database"			"cookies"
	}
```
mysql:
```
	"cookies"
	{
		"driver"			"mysql"
		"host"				"host"
		"database"			"database"
		"user"				"user"
		"pass"				"pass"
		"port"			    "3306"
	}
```
